# Ashraf Ahmed Portfolio Site

This repository contains a static portfolio website intended for GitHub Pages deployment at `https://ashrafahmed9.github.io`.

## What this site includes

- A polished homepage with a strong introduction and call-to-action
- Featured projects with repository links
- A tech slideshow showing skills and tools
- A terminal-inspired code-style section for visual impact
- Contact buttons and a clean, modern layout

## Files

- `index.html` — main portfolio page
- `styles.css` — styling for the landing page
- `script.js` — interactive terminal and tech slideshow logic

## Deploying for free on GitHub Pages

1. Create or use the repository named `ashrafahmed9.github.io`.
2. Place these files in the root of that repository.
3. Commit and push the branch to GitHub.
4. GitHub Pages will publish the site automatically at:
   `https://ashrafahmed9.github.io`

Example commands:

```bash
git init
git add .
git commit -m "Launch portfolio site"
git remote add origin https://github.com/AshrafAhmed9/ashrafahmed9.github.io.git
git branch -M main
git push -u origin main
```

## How to customize

- Replace the hero copy with your own summary.
- Update project cards with your repository and case study links.
- Add a custom photo or a logo in `index.html`.
- Add more tech items in `script.js`.

## Notes

This site is built as a lightweight static portfolio so it loads fast and gives a strong visual signal to recruiters and interviewers. GitHub Pages is free and easy to use, making it ideal for deploying your portfolio quickly.
